0136 Single Number--easy

发表于 2020-02-18 分类于 Algorithm

题目

Given a non-empty array of integers, every element appears twice except for one. Find that single one.

Note: Your algorithm should have a linear runtime complexity. Could you implement it without using extra memory?

Example

自己的解答

主要思想：将nums排序，对nums中的每个元素，查看前一项和后一项是否和自身重复，前后均无重复，则为要找的数。

Runtime: 96 ms, faster than 29.94% of Python3 online submissions for Single Number.

Memory Usage: 15.2 MB, less than 8.20% of Python3 online submissions for Single Number.

解法1: List operation

主要思想：新建一个 no_duplicate_list 列表。遍历nums中的所有元素，如果no_duplicate_list没有，则append，否则remove。

Runtime: 1228 ms, faster than 7.68% of Python3 online submissions for Single Number.

Memory Usage: 15.2 MB, less than 18.03% of Python3 online submissions for Single Number.

时间复杂度：$O(n^2)$。遍历nums中的所有元素需要$O(n)$，搜索no_duplicate_list又需要$O(n)$，故总的时间复杂度为$O(n^2)$。

空间复杂度：$O(n)$。因为有个no_duplicate_list。

解法2: Hash Table

主要思想：用Hash Table避免搜索时所需的$O(n)$的时间复杂度（其实就是用字典替换解法1中用于搜索的list: no_duplicate_list）

Runtime: 92 ms, faster than 44.14% of Python3 online submissions for Single Number.

Memory Usage: 15.3 MB, less than 8.20% of Python3 online submissions for Single Number.

时间复杂度：$O(n)$，因为搜索字典的时间复杂度是$O(1)$

空间复杂度：$O(n)$

解法3: Math

主要思想：$2*(a+b+c)-(a+a+b+b+c)=c$

Runtime: 88 ms, faster than 67.38% of Python3 online submissions for Single Number.

Memory Usage: 15.2 MB, less than 9.84% of Python3 online submissions for Single Number.

时间复杂度：$O(n+n)=O(n)$，sum会调用next函数来遍历nums中的每个元素，等价于sum(list(i, for i in nums))，意味着时间复杂度为$O(n)$

空间复杂度：$O(n)$，set(nums)需要$O(n)$的存储空间。

解法4: Bit Manipulation（位操作）

主要思想：Python中的按位异或（^）操作具有以下性质：

  1. $a\oplus0=a$

  2. $a\oplus{}a=0$

  3. $a\oplus{}b\oplus{}a=(a\oplus{}a)\oplus{}b=0\oplus{}b=b$

    显然$\oplus$满足交换率

利用以上性质即可求解本问题，比如 nums=[4, 1, 2, 1, 2]，$4\oplus1\oplus2\oplus1\oplus2$ $=4\oplus(1\oplus1)\oplus(2\oplus2)$ $=4\oplus0\oplus0=4$

Runtime: 88 ms, faster than 67.38% of Python3 online submissions for Single Number.

Memory Usage: 15.2 MB, less than 9.84% of Python3 online submissions for Single Number.

时间复杂度：$O(n)$，只需要遍历nums即可，时间复杂度就算nums中元素的个数。

空间复杂度：$O(1)$
