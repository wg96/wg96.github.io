如何加入Google？

发表于 2020-02-19 更新于 2020-03-29 分类于 找工作 ， 经验

1. 自信

持之以恒地科学努力。科学的努力是有用的。只要坚持下去，时间是可以改变很多东西的！如果是应届生，还是会看你在学校里的成绩单。

2. 成功入职的流程

投简历 -> 一轮电话面试 -> 大概四轮的技术面试 -> 招聘委员会审核 ->办公地点选择 -> 工作小组双向配对 -> 入职时间确定 -> 入职

3. 准备简历

  • 简历最好是 英文1页 + 中文1页。如果不是在中国office面试，可以不要中文简历。多页亦可。

  • 要简历负责。简历就是面试官考纲

  • 面试地点是你最方便的。当面试全通过了，才会选择工作地点

  • 面试最主要是通过解决算法题，考验你的沟通表达能力、逻辑思维能力、代码实现能力（会要求在”白板”上写代码/伪代码，请练习英语介绍代码实现）

  • 可能会考到系统设计（？？？），会考察你简历上的技能和项目。

  • 英语口语非常重要，我个人觉得，英文好坏对通过面试的影响程度超过30%.

4. 准备和学习

  • 岗位

    Google career

  • 刷题

    1. Leetcode

    2. Google Kickstart 是Google笔试的必然入口，题目质量非常高，推荐刷完所有题目

    3. Google Code Jam 学有余力，继续刷Google Code Jam的题目

    重要的不是题数，而是对编程思维的理解。要注重题目的多样性和难度，可以训练自己在纸上写代码，并且用英文说出来，解释自己是在做什么。这个是非常非常棒的训练方法。

5. 面试技巧

  1. 准备英文自我介绍。

  2. 准备回答英文算法面试题的词汇。

  3. 准备对一些基本数据结构、浅要算法思想（递归、分治、二分等基本算法思想）的理解

  4. 良好的外貌（主要是干净、整洁、有条理）和精神面貌

  5. 好好休息。保证足够的睡眠，和大脑活力、身体体力（面试四轮还是有点累的）
